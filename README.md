# BayraktarYKS

YKS TYT-AYT puanı ve sıralaması hesaplamak için geliştirilmiş açık kaynaklı bir web uygulaması.

🔗 Canlı Demo: https://bayraktaryks.streamlit.app/

## Özellikler

- TYT ve AYT puan hesaplama
- Yerleştirme puanı (OBP) hesaplama
- 2019-2025 ÖSYM verilerine göre sıralama tahmini
- Geçmiş yıllara göre karşılaştırmalı sonuçlar
- Yüzdelik dilim hesaplama

## Kullanılan Veriler

Veriler tamamen ÖSYM tarafından yayımlanan resmi Sayısal Bilgiler raporlarından oluşturulan veri setlerinden elde edilmiştir.
